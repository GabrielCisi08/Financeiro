Conectar ao Supabase

Configure uma vez — todos que acessarem o link verão os mesmos dados em tempo real.

1 Acesse supabase.com → crie conta gratuita → New Project
2 Aguarde ~2 min o projeto iniciar
3 Vá em Settings → API e copie a URL e a anon key
4 Vá em SQL Editor e execute o SQL abaixo para criar a tabela:
create table lancamentos ( id bigint generated always as identity primary key, type text not null, cat text not null, desc text not null, val numeric not null, date date not null, created_at timestamptz default now() ); alter table lancamentos enable row level security; create policy "acesso publico" on lancamentos for all using (true) with check (true);
Publicar no GitHub Pages
1 Acesse github.com → New repository → nome: financeiro → Public → Create
2 Clique em uploading an existing file → arraste o dashboard_financeiro.html
3 Renomeie o arquivo para index.html antes de subir
4 Settings → Pages → Source: Deploy from branch → main → Save
5 Seu link será: https://seu-usuario.github.io/financeiro/
⚠️ Importante: salve as configurações do Supabase ANTES de fazer o upload. As chaves ficam gravadas no arquivo HTML. Não compartilhe o arquivo — só o link do GitHub Pages.
